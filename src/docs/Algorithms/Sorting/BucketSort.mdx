# Bucket Sort

Bucket sort, or bin sort, is a sorting algorithm that works by distributing the elements
of an array into a number of buckets. Each bucket is then sorted individually, either
using a different sorting algorithm, or by recursively applying the bucket sorting
algorithm. It is a distribution sort, a generalization of pigeonhole sort, and is a cousin
of radix sort in the most-to-least significant digit flavor. Bucket sort can be
implemented with comparisons and therefore can also be considered a comparison sort
algorithm. (Source: Wikipedia)
